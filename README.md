# EVENTIFI - Site de Eventos

## Sobre o Projeto

EVENTIFI é uma plataforma web para gerenciamento e divulgação de eventos. Desenvolvida com PHP e Bootstrap, oferece uma experiência intuitiva para organizadores e participantes.

## Funcionalidades

- Cadastro e gerenciamento de eventos

- Sistema de autenticação de usuários

- Página de detalhes do evento

- Busca e filtros de eventos

- Área administrativa

- Responsividade para dispositivos móveis

## Tecnologias Utilizadas

- PHP: 8.3.6

- Composer: 2.9.5

- Front-end: Bootstrap 5.3.8

- Roteamento: pecee/simple-router
